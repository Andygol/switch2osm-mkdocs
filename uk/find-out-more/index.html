    Отримання допомоги⚓︎

    OpenStreetMap – децентралізований проєкт, який працює на волонтерських засадах, тож важко надати єдину адресу е-пошти для звʼязку. Натомість знання всієї спільноти допоможуть вам розвʼязати будь-яку проблему.

    • OpenStreetMap IRC (Internet Relay Chat): цілодобовий живий чат OSM (канали #osm або #osm-dev)
    • Форум спільноти
    • Часті питання для розробників
    • Часті питання (загальні)

    Інші довідники щодо переходу на OSM⚓︎

    • Настанови по картографії від Kelso
    • Розгортаємо сервер з використанням MapServer

    Про switch2osm⚓︎

    Switch2osm – це сайт, керований спільнотою, який координується Річардом Ферхерстом (Richard Fairhurst) та Полом Норманом (Paul Norman) з допомогою спільноти OpenStreetMap. Ви також можете взяти участь у покращенні цього сайту на GitHub (англ. версія). Переклад українською виконано українською спільнотою. Виправлення помилок, пропозиції щодо покращення тексту завжди вітаються, чекаємо їх також на GitHub (osm-ua).

    Маєте пропозиції або коментарі? Звʼяжіться з авторами в IRC.

    Бажаєте поспілкуватись з українською спільнотою: звертайтесь на форум, в Телеграм або Slack.
